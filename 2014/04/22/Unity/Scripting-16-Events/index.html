How to create a dynamic “broadcast” system using Events.

事件是一种特殊的委托（delegate），可以通知其他类某个特定的状况发生了。

事件可以看做是一个广播系统，任何对该事件感兴趣的类可以通过方法订阅它，那么在某种情况发生了，例如点击了按钮，玩家掉血，我们会调用事件，与此同时会调用所有订阅该事件的方法。
